# 프론트엔드 (React)

## 프로젝트 구조

- React 기반 SPA
- JWT 토큰을 활용한 인증 처리

## 주요 라이브러리

- React
- Zustand
- Styled-components
- React Hook Form
- Axios
- React Calendar / Datepicker / Time Picker
- React Toastify
- React Spinners
- Vite

## 실행 방법

```bash
cd front
npm install
npm start
```
